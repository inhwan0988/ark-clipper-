# ARK Clipper 사용법

YouTube 롱폼 영상에서 후킹되는 부분을 AI로 찾아 자동으로 쇼츠를 만들어주는 **로컬 도구**입니다.

각자 본인 컴퓨터에 설치해서 사용합니다. (인터넷 공유 X)

---

## 📦 1회 설치 (처음 한 번만)

### 사전 준비
1. **Node.js 설치**: https://nodejs.org → LTS 버전 다운로드 → 설치
2. **FFmpeg 설치**: PowerShell 관리자 권한으로 실행 후
   ```powershell
   winget install Gyan.FFmpeg
   ```
   설치 후 **컴퓨터 재시작 또는 로그아웃 → 다시 로그인**
3. **NVIDIA GPU 권장** (없어도 동작하지만 음성 인식이 매우 느림)

### 자동 설치
1. `install.bat` **더블클릭**
2. 보안 경고 뜨면 "추가 정보" → "실행" 또는 "예"
3. 자동으로 다음을 설치:
   - Python 3.12 (없으면)
   - Python 가상환경 + faster-whisper
   - yt-dlp.exe
   - npm 패키지
4. 완료 메시지 확인 후 창 닫기

### Anthropic API 키 발급 (1회)
1. https://console.anthropic.com 가입
2. 결제 카드 등록 + $5~$10 충전 (영상 1개 분석에 약 $0.05~0.20)
3. **Settings → API Keys → Create Key**
4. `sk-ant-api03-...` 형태의 키 복사 (한 번만 보임)

---

## ▶️ 실행

### 방법 1: 더블클릭 (가장 쉬움)
1. `start.bat` **더블클릭**
2. 검은 창이 뜨고 약 5초 후 **브라우저가 자동으로 열림**
3. 브라우저에서 **우상단 API 키 입력** (1회)
4. 사용 시작!

### 방법 2: 직접 실행
```
npm run dev
```
브라우저로 http://localhost:3000 접속

### 종료
- 검은 창에서 **Ctrl + C**
- 또는 창 닫기

---

## 🎬 사용 흐름

1. **API 키 입력** (홈 화면, 1회만)
2. **저장 폴더 설정** (선택, 비워두면 기본 폴더)
3. **YouTube URL 붙여넣기** → "쇼츠 만들기"
4. **영상 미리보기** 확인 → "맞아요, 분석 시작"
5. **자동 처리** (다운로드 → 음성 인식 → AI 분석)
6. **AI 추천 후킹 구간 5~6개** 검토
   - 좌우 화살표로 영상 넘기기
   - 영상 미리 보면서 시간 조정
   - 제목 / 자막 / 채널명 디자인 커스텀
7. **클립 생성** → 자동으로 9:16 쇼츠 만들어짐
8. **MP4 다운로드** 또는 **전체 ZIP 다운로드**

## ❓ 자주 묻는 질문

### Q. API 키는 매번 입력해야 하나요?
**아니요.** 한 번 저장하면 그 브라우저에서는 영구 저장됩니다.
다른 브라우저나 다른 PC에서 사용하려면 거기서 한 번 더 입력.

### Q. 영상이 어디에 저장되나요?
기본: `[ARK_Clipper 폴더]\workspace\[프로젝트id]\clips\`
원하는 폴더로 변경하려면 홈 화면 "저장 폴더" 설정.

### Q. API 키와 비용은 안전한가요?
- API 키는 본인 브라우저에만 저장됨 (서버나 외부 공유 X)
- 비용은 본인 Anthropic 계정에서 차감
- 영상 1개 분석 = 약 $0.05~$0.20 (영상 길이에 따라)

### Q. 인터넷 없이 사용할 수 있나요?
- 영상 다운로드 + AI 분석은 인터넷 필요
- 음성 인식 + 영상 편집은 오프라인 (로컬 GPU/CPU 사용)

### Q. 다른 사람에게 링크 공유할 수 있나요?
**불가.** 각자 본인 컴퓨터에 설치해서 사용하는 로컬 도구입니다.
공유하려면 이 폴더 전체를 압축해서 전달:
1. ARK_Clipper 폴더 압축 (.env.local 제외)
2. 받는 사람이 압축 풀고 install.bat 실행
3. 받는 사람도 본인 API 키 발급 + 입력

### Q. 음성 인식이 너무 느려요
- NVIDIA GPU 없으면 large-v3 모델이 매우 느립니다 (1시간 영상에 30~60분)
- GPU 있으면 동일 영상에 3~5분
- GPU 드라이버 + CUDA 라이브러리 설치 확인

### Q. 에러가 나요
- 검은 창의 빨간색 메시지 확인
- `start.bat` 다시 실행
- 그래도 안 되면 `install.bat` 다시 실행

## 🛠️ 고급

### 저장 위치 옮기기
홈 화면 "저장 폴더" 칸에 절대 경로 입력:
- 예: `D:\ARK_Shorts`
- 예: `C:\Users\사용자\Documents\Shorts`

### 기존 영상 다시 분석
프로젝트 페이지에서 **🔄 AI 다시 분석** 버튼 클릭.
새 후킹 구간 + 제목이 다시 생성됩니다 (재전사 X, API 비용만).

### 동시 사용 (한 PC, 여러 명)
- 같은 OS 사용자 계정 + 같은 브라우저 = 키 공유됨
- 다른 사람 사용 시: 브라우저 프로필 분리 (Chrome 사람 추가)
